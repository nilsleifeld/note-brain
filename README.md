# NoteBrain

A small web app for **practicing note reading** in **treble clef** (G clef). A random note appears on the staff; you pick the correct label from several choices. After you answer, you see what was right — **Next** starts another round.

## Requirements

- [Bun](https://bun.sh) (dev server with hot reload)

## Getting started

```bash
bun install
bun run dev
```

The server runs with hot reload from `src/index.html` — check the terminal for the exact URL (often `http://localhost:3000` or similar).
